Abstract

For a fixed k𝑘k-uniform family ℱℱ{\cal{F}} of hypergraphs, the linear Turán number exlin​(n,ℱ)subscriptexlin𝑛ℱ{\rm ex}_{\rm lin}(n,{\cal{F}}) is the maximum number of edges in a k𝑘k-uniform linear hypergraph ℋℋ\mathcal{H} on n𝑛n vertices that does not contain any member of ℱℱ{\cal{F}} as a subhypergraph.

Moreover, |ℋ|=n2/k2ℋsuperscript𝑛2superscript𝑘2|{\mathcal{H}}|=n^{2}/k^{2} holds if and only if n≡0(modk)𝑛annotated0pmod𝑘n\equiv 0\pmod{k} and ℋℋ\mathcal{H} is a transversal design on n𝑛n points with k𝑘k groups.

1 Definitions, the linear Turán number

A k𝑘k-uniform hypergraph ℋ=(V,E)ℋ𝑉𝐸{\cal{H}}=(V,E) has vertex set V𝑉V and its edge set E𝐸E consists of some k𝑘k-element subsets of V𝑉V (repeated edges are not allowed). A hypergraph is k𝑘k-partite if its vertices can be partitioned into k𝑘k groups so that each edge has exactly one vertex from each group.

For a fixed k𝑘k-uniform family ℱℱ{\cal{F}}, the Turán number ex​(n,ℱ)ex𝑛ℱ{\rm ex}(n,{\cal{F}}) is the maximum number of edges in a k𝑘k-uniform hypergraph ℋℋ\mathcal{H} of n𝑛n vertices that is ℱℱ\cal{F}-free, i.e., ℋℋ\mathcal{H} does not contain any member of ℱℱ{\cal{F}} as a subhypergraph. The systematic study of Turán numbers for graphs was started with the paper of Turán [13] determining ex​(n,Kt)ex𝑛subscript𝐾𝑡{\rm ex}(n,K_{t}). The case ex​(n,K3)=⌊n2/4⌋ex𝑛subscript𝐾3superscript𝑛24{\rm ex}(n,K_{3})=\lfloor{n^{2}/4}\rfloor was discovered earlier [10] and usually cited as Mantel’s theorem. Turán [14] also initiated extensions for hypergraphs, the surveys [6]–[9] are covering many developments.

A hypergraph is called linear if any two edges intersect in at most one vertex. Notice that 222-uniform linear hypergraphs are graphs. In design theory 333-uniform linear hypergraphs are called partial triple systems and fixed small ones are called configurations. See the seminal monograph by Colbourn and Rosa [4].

For a fixed family of hypergraphs ℱℱ{\cal{F}}, the linear Turán number exlink​(n,ℱ)superscriptsubscriptexlin𝑘𝑛ℱ{\rm ex}_{\rm lin}^{k}(n,{\cal{F}}) is the maximum number of edges in a k𝑘k-uniform linear hypergraph ℋℋ\mathcal{H} on n𝑛n vertices that is ℱℱ\cal{F}-free, i.e., ℋℋ\mathcal{H} does not contain any member of ℱℱ{\cal{F}} as a subhypergraph. The k𝑘k-uniform linear ℱℱ\cal{F}-free hypergraphs ℋℋ\mathcal{H} on n𝑛n vertices with exlink​(n,ℱ)superscriptsubscriptexlin𝑘𝑛ℱ{\rm ex}_{\rm lin}^{k}(n,{\cal{F}}) edges are called extremal hypergraphs. The upper index k𝑘k is often omitted.

Linear Turán numbers studied mostly implicitly. For example, the upper bound of Ruzsa and Szemerédi [12] on triple systems not carrying three edges on six vertices is equivalent to exlin3​(n,T)=o​(n2)superscriptsubscriptexlin3𝑛𝑇𝑜superscript𝑛2{\rm ex}_{\rm lin}^{3}(n,T)=o(n^{2}) where T𝑇T is the linear triangle (a configuration {123,345,561}123345561\{123,345,561\}). A recent remarkable work on this topic is [5] where the linear Turán numbers of k𝑘k-uniform linear cycles were studied.

1.1 k𝑘k-fans and transversal designs

In this paper we study the linear Turán number exlin​(n,Fk)subscriptexlin𝑛superscript𝐹𝑘{\rm ex}_{\rm lin}(n,F^{k}) where Fksuperscript𝐹𝑘F^{k}, the k𝑘k-fan, is defined as follows. For k≥2𝑘2k\geq 2, Fksuperscript𝐹𝑘F^{k} is the linear k𝑘k-uniform hypergraph having k+1𝑘1k+1 edges f1,…,fksubscript𝑓1…subscript𝑓𝑘f_{1},\dots,f_{k} and g𝑔g such that f1,…,fksubscript𝑓1…subscript𝑓𝑘f_{1},\dots,f_{k} all contain the same vertex v𝑣v (the center) and the additional crossing edge g𝑔g intersects all fisubscript𝑓𝑖f_{i} in a vertex different from v𝑣v. Note that F2superscript𝐹2F^{2} is the graph triangle and F3superscript𝐹3F^{3} (sometimes called a sail) is configuration C15subscript𝐶15C_{15} in the list of small configurations in [4].

A transversal design T​(n,k)𝑇𝑛𝑘T(n,k) on n𝑛n vertices with k𝑘k groups is a k𝑘k-partite hypergraph with groups of equal size (thus n𝑛n is a multiple of k𝑘k) and each pair of vertices from different groups is covered by exactly one hyperedge. It is well-known that transversal designs T​(n,k)𝑇𝑛𝑘T(n,k) (or equivalently k−2𝑘2k-2 Mutually Orthogonal Latin Squares, MOLS) exist for all n𝑛n when n>n0​(k)𝑛subscript𝑛0𝑘n>n_{0}(k) and k𝑘k divides n𝑛n. A truncated design is obtained from a transversal design by removing one vertex (and all edges containing it).

Our main result is the following generalization of Mantel’s theorem.
